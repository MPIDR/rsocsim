# Create a basic .sup file for a simulation

The simulation is only a simple one. The file will be saved into the
directory 'simdir'.

## Usage

``` r
create_sup_file(simdir, simname)
```

## Arguments

- simdir:

  A string. The directory where the .sup file will be saved.

- simname:

  A string. The name of the simulation.

## Value

A string. The filename of the supervisory file which is needed to start
the simulation.
